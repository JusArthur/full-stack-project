# Team Hortons - Style Guide v1.0 (Tim Hortons-inspired)

## Styling Approach
We use **Tailwind CSS** for most component styling.  
`index.css` and `App.css` are kept minimal for global defaults and layout only.

## Colour Palette (3-5)
- **Tim Red (Primary):** `#C8102E`
- **Coffee Brown (Text/Headings):** `#3B2316`
- **Cream (Background):** `#F7F3E9`
- **White (Surface):** `#FFFFFF`
- **Warm Gray (Borders/Muted):** `#E6E0D8`

## Typography
**Primary font stack:** system-ui, -apple-system, "Segoe UI", Roboto, Arial, sans-serif  
**Style notes:** bold headings, clean body text, lots of white space.

## Font Sizes (min 3)
- **Heading 1:** 48px (3rem) - page titles
- **Heading 2:** 32px (2rem) - section titles
- **Body:** 16px (1rem)
- **Small:** 14px (0.875rem)

## UI Guidelines
- Use **Tim Red** for the top navigation background.
- Use **Coffee Brown** for headings and key text.
- Use **Cream** as the page background.
- Use **White** for buttons and surfaces.
- Keep spacing generous; avoid heavy shadows.
